Полный цикл работы sphinx:
1. Запуск search/indexing.json. 
	1.а) Исходные данные:
		1.a.1)indexing.json с указанием пути к индексируемым объектам; 
		1.a.2)файл индексов search/sys/rwindex с готовыми данными (индексами deltaindex, mainindex и конфигурацией) или восстановленный из шаблона search/sys/rwindex_copy;
		1.a.3)файл хранящий ID индексируемых документов search/sys/xml_maxindex.data (пустой или с данными);	
		1.a.4)nexes - filesender.nexe, pdf.nexe, doc.nexe, txt.nexe, other.nexe, xmlpipecreator.nexe, indexer.nexe 
	1.b) Выполняемые действия:
		1.b.1)запускает кластер из обьектов находящихся по указанному пути;
		1.b.2)извлекает метаданные предоставляемые swift из обьектов. А также текстовые данные из документов формата doc, docx, pdf, txt, odt;
		1.b.3)проводит индексацию полученных данных;
		1.b.4)результат сохраняется в обьекте serach/sys/rwindex (перезаписываются только данные deltaindex);

2. Запуск merge.json
	2.а) Исходные данные:
		2.a.1)merge.json; 
		2.a.2)файл индексов search/sys/rwindex с готовыми данными (индексами deltaindex, mainindex и конфигурацией);
		2.a.2)nexes - indexer.nexe 
	2.b) Выполняемые действия:
		2.b.1)распаковывает deltaindex, mainindex а также конфигурацию из объекта search/sys/rwindex
		2.b.2)проводит объединение deltaindex => mainindex;
		2.b.3)результат сохраняется в обьекте serach/sys/rwindex (обновляются только данные mainindex);

3. Запуск search.json
	3.а) Исходные данные:
		3.a.1)search.json;
		3.a.2)файл индексов search/sys/rwindex с готовыми данными;
		3.a.2)nexes - search.nexe
	3.b) Выполняемые действия:
		3.b.1)распаковывает deltaindex, mainindex а также конфигурацию из объекта search/sys/rwindex
		3.b.2)проводит поиск по mainindex;
		3.b.3)результат выдает в stdout;




примеры запросов вводимых пользователем:
request: "example text "CONTENT_TYPE:application/msword!@#$%^&*(){}|\/~`<>,.+-" CONTENT_LENGTH:1000" 
args: "-c index/zsphinx.conf -i mainindex -w -m example text CONTENT TYPE application msword CONTENT LENGTH 1000 -j CONTENT_TYPE application/msword@#$%^&*(){}|\/~`<>,.+- -j CONTENT_LENGTH 1000"

request: "example text CONTENT_TYPE application/msword"
args: "-c index/zsphinx.conf -i mainindex -w -m example text CONTENT TYPE application msword"
